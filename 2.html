    <ul>OrderedList tag</ul> : 부모 태그로 해당 태그 안에 ListItem 태그를 넣으면 앞에 공백과 목록 숫자를 만들어준다.

    CSS

    <h1>heading tag</h1> : h1~h6까지 제목을 6단계의 크기로 나눌 수 있으며 1이 제일 크고 6이 제일 작다.

    strong
    <strong>strong tag</strong> : 글자를 굵게 만들어주는 태그.

    u
    <u>underline tag</u> : 글자에 밑줄을 그어주는 태그.

    p
    <p>parameter tag</p> : 문장의 단락을 만들어주는 태그.

    <p style="margin-top:45px;">parameter tag</p> CSS용법으로 해당 단락과 위 단락을 45픽셀 만큼의 거리를 벌려 줌.

    소레요리노 전주시
    <a href="링크할 주소" target="_blank" title="링크에 대면 나오는 설명(제목)"> : a = Anchor 닻,href = hypertext reference, target="_blank" = 링크를 누를 시 새 탭으로 열린다.


    <img src="이미지 파일명" width="크기%"> : img는 이미지 태그고 src="이미지 파일명" width="크기%"는 attribute(속성)으로 순서는 상관이 없다.
